Get your bones up and running.

![](skeletons.gif)

```bash
$ npm i
$ npm run build
```

Then, in a separate terminal window or tab:
```bash
$ npm start
```

Now visit [`http://localhost:13000/`](http://localhost:13000/) in your browser
